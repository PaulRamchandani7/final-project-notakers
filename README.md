# NoTaker API

This is the backend API for the NoTaker application.

## API Documentation

The API documentation is available via Swagger UI.

1.  Start the server: `npm start` (or `npm run dev`)
2.  Open your browser and navigate to: `http://localhost:3000/api-docs`

## Features

-   **Authentication**: User signup and login with JWT.
-   **Users**: Manage user profiles and roles.
-   **Notes**: Create, read, update, and delete notes.
-   **Subjects**: Organize notes by subject.
-   **Study Reminders**: Set reminders for study sessions.

## Tech Stack

-   Node.js
-   Express.js
-   Prisma (ORM)
-   PostgreSQL (Database)
-   Swagger/OpenAPI (Documentation)
